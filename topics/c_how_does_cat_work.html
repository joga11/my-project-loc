How does CAT work?

About CAT concept

First, CAT tool breaks the source text into translation units or segments (TUs). Usually, TUs correspond to sentences or similar wholes. Then, the tool creates a translation memory (TM) for storing the source and translated segments. The CAT search engine allows the translator to view saved segments, search for segments with similar content, and translate them.

With the help of termbases, glossaries, and translation memories, the CAT tool helps the translator ensure the terminology and writing style of the source file are translated accurately and as intended.

Depending on available options, CAT tools include such features like autosuggest, quality assurance, side-by-side alignment, optionally feature of the Machine Translation service, and much more. Use a spell checker, manage terms, use specific dictionaries, term databases, text indexers, merge the translation and the source text, manage projects, translation memories, and automate translation work in general.
